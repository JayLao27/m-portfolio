# Webisayt – Personal Portfolio

A personal portfolio website built with React, TypeScript, and Vite, focused on clean design, smooth interactions, and performance. The site showcases selected projects and experiments in web development, machine learning, and IoT.

## Technologies

- **Frontend:** React 19, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animation & Motion:** GSAP, Locomotive Scroll
- **3D & Visual Effects:** Three.js (Drei, Postprocessing)

## Features

- **Dark / Light Mode**  
  Simple theme switching with custom color tokens.

- **Interactive UI**  
  Subtle hover effects, cursor interactions, and motion-based feedback.

- **Responsive Layout**  
  Designed to work across desktop, tablet, and mobile devices.

- **Project Showcase**  
  Displays selected projects related to web development, machine learning, and IoT.

- **Visual Enhancements**  
  Includes grain textures and animated gradients for added depth without heavy visuals.

- **AI Chatbot Assistant**  
  Features JayBot, an AI-powered conversational clone of Jay Lao, built with local RAG retrieval fallback and Groq streaming API integration.
